## 1. 제품 포지셔닝

### 핵심 가치 제안
- **기존 ERP 대체 ❌ → 보완 ✅**
- 학생 데이터 기반 학습 분석 전문 도구
- 학부모 상담 자료로 즉시 활용 가능
- 데이터 기반 의사결정으로 재등록률 향상

### 차별화 포인트
| 기능 | StudyCoach | 기존 학원 ERP |
|------|------------|---------------|
| 능력치 레이더 차트 | ✅ 핵심 기능 | ❌ 없음 |
| 성적 추세 분석 | ✅ AI 기반 | ⚠️ 단순 그래프 |
| 능력 계산 엔진 | ✅ 독자 알고리즘 | ❌ 없음 |
| 학습 패턴 분석 | ✅ 예정 | ❌ 없음 |
| 결제/청구 | ❌ 없음 | ✅ 핵심 |
| 출결 관리 | ⚠️ 기본 | ✅ QR/NFC |

### 타겟 마케팅 메시지
> "성적표만 올리면, 학생별 맞춤 학습 전략이 나옵니다"
> "데이터로 증명하는 학원의 교육 효과"

## 2. 기술적 준비사항

### 현재 기술 스택
- **프론트엔드**: Next.js 15.5.4 (App Router)
- **인증**: NextAuth.js v5 beta
- **데이터베이스**: Prisma 5.22.0 + SQLite
- **차트**: Recharts
- **스타일링**: TailwindCSS + shadcn/ui

### 즉시 필요한 작업
| 항목 | 현재 상태 | 필요 작업 | 우선순위 |
|------|----------|----------|----------|
| 멀티테넌시 | Academy 모델 ✅ | 데이터 격리 강화 | P1 |
| 결제 시스템 | 없음 ❌ | Toss/Stripe 연동 | P2 |
| 사용량 추적 | 없음 ❌ | API 호출, 학생 수 측정 | P2 |
| 온보딩 플로우 | 기본 ⚠️ | 셀프서비스 가입, 데모 데이터 | P1 |
| DB 마이그레이션 | SQLite | PostgreSQL (Supabase) | P2 |

### 스케일링 로드맵
```
Phase 1: SQLite (개발/테스트)
    ↓
Phase 2: PostgreSQL - Supabase (초기 상용화)
    ↓
Phase 3: PostgreSQL + Redis 캐싱 (성장기)
    ↓
Phase 4: 분산 DB + CDN (대규모)
```

## 4. 고객 유치 전략 (Go-to-Market)

### Phase 1: 초기 고객 확보 (0-100)
| 채널 | 전략 | 예상 CAC |
|------|------|---------|
| 학원 커뮤니티 | 맘카페, 학원장 네이버 카페 | ₩0-5,000 |
| 무료 체험 | 14일 무료 + 분석 리포트 | ₩0 |
| 레퍼럴 | 추천 시 1개월 무료 | ₩15,000 |
| SEO 콘텐츠 | "성적 분석법" 블로그 | ₩0 |

### Phase 2: 성장 (100-1,000)
- 교육 컨설팅 업체 파트너십
- 학원 협회 연계 마케팅
- 성공 사례(Case Study) 홍보

### Phase 3: 스케일 (1,000+)
- 유료 광고 (네이버, 카카오)
- 기존 ERP 업체와 API 연동
- 프랜차이즈 학원 B2B 영업

## 5. 고객 관리 (Customer Success)

### 온보딩 자동화
```
Day 1:  가입 → 데모 데이터 자동 생성 → 핵심 기능 투어
Day 3:  이메일 "첫 학생 등록하셨나요?" + 가이드
Day 7:  첫 분석 리포트 생성 유도
Day 14: 유료 전환 안내 + 할인 제안
```

### 이탈 방지 모니터링
| 지표 | 위험 신호 | 자동 대응 |
|------|----------|----------|
| 7일 미접속 | 🚨 High | 리마인드 이메일 |
| 분석 0회/월 | 🚨 High | 1:1 온보딩 콜 제안 |
| 학생 수 감소 | ⚠️ Medium | 피드백 설문 |

### 핵심 KPI
- **MRR** (Monthly Recurring Revenue): 목표 ₩10M/월 (1년차)
- **Churn Rate**: 목표 <5%/월
- **NPS**: 목표 40+
- **Time to Value**: 첫 분석까지 <30분

## 7. 경쟁 분석

### 시장 현황
| 서비스 | 강점 | 약점 | 가격 |
|--------|------|------|------|
| 랠리즈 | 종합 ERP | 분석 기능 약함 | ₩50-200K |
| 클래스업 | 모바일 | 데이터 분석 없음 | ₩30-100K |
| 어나더클래스 | UI/UX | 학습 분석 없음 | ₩40-150K |
| 에듀허브 | 통합 | 복잡함 | ₩60-200K |

### StudyCoach 경쟁 우위
1. **학습 분석 전문**: 유일한 AI 기반 학습 분석
2. **낮은 진입장벽**: 기존 ERP와 병행 사용
3. **명확한 ROI**: 학부모 상담 품질 향상
4. **합리적 가격**: 분석 기능만 제공하므로 저렴
